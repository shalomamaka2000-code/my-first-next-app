# 🌸 Rosy Flower Shop – Flower Shop Landing Page

A modern and responsive flower shop landing page built with Next.js.  
This project showcases elegant floral products, featured collections, and a beautiful user-friendly interface designed to provide a seamless
browsing experience.

---

## ✨ Features

- Responsive modern UI
- Hero section
- Featured flower collections
- Product showcase cards
- Smooth hover animations
- Mobile-friendly layout
- Clean and elegant design
- Fast performance with Next.js

---

## 🛠️ Tech Stack

- Next.js
- React
- TypeScript
- Tailwind CSS
